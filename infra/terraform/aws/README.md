# AWS EKS GPU test stack

Provisions the AWS EKS GPU-ready cluster used by the keda-gpu-scaler e2e
tests. See [`infra/terraform/README.md`](../README.md) for the cross-cloud
index, quota, cost, and teardown notes, and [`bootstrap/`](./bootstrap) for
the run-once remote-state + OIDC setup.

The inputs/outputs table below is generated by `terraform-docs` in CI — do not
edit it by hand.
